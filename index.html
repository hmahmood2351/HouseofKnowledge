House of Knowledge

About

بسم الله الرحمن الرحيم

This web application is aimed to be a quick access repository of Islamic knowledge and a web scraper to retrieve the latest prayer times, without any intrusive advertisements or pop-ups. More features are planned for the future, إن شاء الله

Some of the planned features:

  • Web Scraper
  • PDF excerpts of invocations
  • Night/day functionality
  • Interactive web scraper
  • Interactive invocations
  • Integrated social media functionality
  • Mobile device compatibility (Android only for now)
